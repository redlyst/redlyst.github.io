Online Laundry

On demand mobile simulation application for connecting laundry service with the customer.

  • A Mobile Application build with Java-Native language using Android Studio
  • A Backend Web Application and Application Programmming Interface (API) build with PHP-Native language
  • MySQL as Database
  • Firebase Cloud Messaging for notification order
  • Google Maps API Service for maps with marker and polylines on android
  • A* (A-Star) Algorithm for maps direction and route cost
